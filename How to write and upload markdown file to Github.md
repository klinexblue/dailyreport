# Markdown 문서 양식(글 꾸미는 방법)
md 파일에서는 특정한 문자(열)를 특정 블럭으로 에워 마크업 효과를 쓸 수 있습니다.  
그러나 현재 md 파일의 표준이 존재하지 않아 양식과 효과가 플랫폼별로 난립하는 실정입니다.  

본 문단에서는 Github에서 실제로 유효한 효과만을 소개하였습니다.  
읽으시는 분들이 보다 효과적으로 내용을 인식할 수 있도록, 시의적절히 이용합시다.

## 줄 바꾸기

일반적인 텍스트 파일처럼 그냥 엔터 글쇠 한번만 치겠습니다.
줄바꿈이 전혀 이뤄지지 않습니다. 그러므로 엔터 글쇠를 치기 전에 띄어쓰기를 두 번 넣고 엔터키를 쳤습니다.  

드디어 줄이 바뀌었습니다.  
```
일반적인 텍스트 파일처럼 그냥 엔터 글쇠 한번만 치겠습니다.
줄바꿈이 전혀 이뤄지지 않습니다. 그러므로 엔터 글쇠를 치기 전에 띄어쓰기를 두 번 넣고 엔터키를 쳤습니다.  

드디어 줄이 바뀌었습니다.  
```

md 파일을 작성하실 때 기본적으로 알아야 하는 내용입니다.  
줄을 바꾸려는 부분을 Enter 키로 치기 전에 띄어쓰기를 두 번 넣습니다.  

띄어쓰기 두 번을 잊고 Enter 키로만 줄을 바꾸려 드시면 줄이 바뀌지 않습니다.  

헤드라인의 경우 띄어쓰기 없이 Enter 키만 쳐도 줄이 바뀌며, Enter 글쇠를 누른 부분 다음은 헤드라인으로 취급되지 않습니다.

## 문자열에 쓸 수 있는 마크업 효과: 기울임체·진하게·취소선

*기울임체*  
**진하게**  
_기울임체_  
__진하게__   
~~취소선~~  

```
*기울임체*  
**진하게**   
_기울임체_  
__진하게__  
~~취소선~~  
```

마크업 효과를 주려는 문자열의 양 옆으로 아스타리스크(*) 또는 언더바(_), 물결표(~) 등을 에웁니다.  
효과를 주려는 영역 이외의 문자열과는 띄어쓰기를 넣어 구분합니다.

```
# 제목 1단계
## 제목 2단계
### 제목 3단계
#### 제목 4단계
##### 제목 5단계
###### 제목 6단계
####### 헤드라인 효과 없음
```

제목에 크기 차이 및 위계를 주어 글을 문단별로 읽기 좋게 합니다.  
해시 또는 샤프(#)글쇠를 1개부터 6개까지 잇달아 쓴 뒤 띄어쓰기를 하고 넣으려는 제목을 씁니다.

```

```

기본적으로 양식을 갖춘 홈페이지 주소, 이메일 주소 등 도메인 네임은 별도의 처리 없이 자동으로 해당 페이지로 연결됩니다.  
도메인 네임 바로 다음 다른 문자열이 끼면 띄어쓰기가 나올 때까지 도메인 네임의 연장으로 인식합니다.  
그러므로 도메인 네임을 인용한 후에 반드시 띄어쓰기 한 번으로 도메인 네임을 구분시킵시다.

그 옆에 소괄호로 감싼 도메인 네임을 넣으면 해당 문자열을 눌렀을 때 의도한 페이지로 연결시킬 수 있습니다.  

만약 도메인 네임에 하이퍼링크가 자동 연결되는 것을 원치 않으신다면,  
하이퍼링크가 못 생성되게 할 도메인 네임을 억음 부호 글쇠(`)로 에웁니다.

문단을 나눠 읽기 좋게끔 가로줄을 긋습니다. 아스타리스크(*)나 대시(-), 등호 (=)글쇠를 세 번 이상 연타하고 줄을 바꿉니다.  
바로 앞 줄에 문자열이 있다면 앞 줄의 문자열을 제목으로 인지하게 되므로, 문자열 다음에 가로줄을 긋고자 할 때는 줄을 두 번 이상 바꿉니다.

```

문장 도중 별도의 해설이 필요하나 본문에 넣기 너무 긴 경우  
본문 내에 간단한 앵커만 남기고, 별도 해설을 다른 공간에 넣는 기능입니다.  

본문에는 대괄호([])안에 곡절 부호(^)를 넣고 그 다음 숫자 또는 별도의 문자열을 넣어 앵커를 만듭니다.  

그리고 해설이 본문에 방해되지 않을 공간에 앵커와 똑같은 구분자를 쓴 뒤 콜론(:)을 쓰고 띄어쓰기하고, 해설을 씁니다.  
해설이 길어질 경우 줄을 바꾸고 다음 줄에서 띄어쓰기를 네 번 하여 주석을 이어나갈 수 있습니다.  
실제 해설은 본문에서 맨 마지막 줄에 가게 되며, 본문과 해설에 각각 상대 위치로 이동하는 앵커가 생깁니다.

```

| ---- | ---- |

| :--- | :---: | ---: |
| Left | Center | Right |
| 左 | 中 | 右 |
```

표를 만듭니다. 줄 구분은 줄바꿈으로, 칸 구분은 파이프 글쇠(|)로 합니다.  
표제가 될 줄 바로 다음 줄을 대시(-)글쇠 3회 이상 연타로 채우면 표제 줄 문자열이 자동으로 진해집니다. 내용은 그 다음 줄부터 채웁니다.   

한 열의 정렬 방향을 표제 줄과 맞출 수도 있습니다.  
대시(-)글쇠를 연타한 부분의 어느 한쪽으로 세미콜론 글쇠(:)가 있으면 그 방향으로 정렬됩니다.  
세미콜론 글쇠로 대시 연타 문자열 양쪽을 에우면 가운데로 정렬됩니다.

- [ ] 엄마한테 칭찬받기
- [ ] 좋아하는 사람에게 액세서리 선물하기
```

현재 처리가 완료됐는지 아닌지를 파악하는 체크 리스트를 만듭니다.  
첫 글쇠로 대시(-)를 누르고 띄어쓰기한 뒤, 대괄호([])를 씁니다.  
대괄호 안에 소문자 x 글쇠를 넣으면 체크 박스가 차고, 띄어쓰기하면 체크 박스가 빕니다. 
대괄호 다음 한 번 더 띄어쓰기한 다음 상세내용을 적습니다.
