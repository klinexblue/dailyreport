# Markdown 문서 양식(글 꾸미는 방법)
md 파일에서는 특정한 문자(열)를 특정 블럭으로 에워 마크업 효과를 쓸 수 있습니다.  
그러나 현재 md 파일의 표준이 존재하지 않아 양식과 효과가 플랫폼별로 난립하는 실정입니다.  

본 문단에서는 Github에서 실제로 유효한 효과만을 소개하였습니다.  
읽으시는 분들이 보다 효과적으로 내용을 인식할 수 있도록, 시의적절히 이용합시다.  

## 줄 바꾸기

일반적인 텍스트 파일처럼 그냥 엔터 글쇠 한번만 치겠습니다.
줄바꿈이 전혀 이뤄지지 않습니다. 그러므로 엔터 글쇠를 치기 전에 띄어쓰기를 두 번 넣고 엔터키를 쳤습니다.  

드디어 줄이 바뀌었습니다.  
```
일반적인 텍스트 파일처럼 그냥 엔터 글쇠 한번만 치겠습니다.
줄바꿈이 전혀 이뤄지지 않습니다. 그러므로 엔터 글쇠를 치기 전에 띄어쓰기를 두 번 넣고 엔터키를 쳤습니다.  

드디어 줄이 바뀌었습니다.  
```

md 파일을 작성하실 때 기본적으로 알아야 하는 내용입니다.  
줄을 바꾸려는 부분을 Enter 키로 치기 전에 띄어쓰기를 두 번 넣습니다.  

띄어쓰기 두 번을 잊고 Enter 키로만 줄을 바꾸려 드시면 줄이 바뀌지 않습니다.  

헤드라인의 경우 띄어쓰기 없이 Enter 키만 쳐도 줄이 바뀌며, Enter 글쇠를 누른 부분 다음은 헤드라인으로 취급되지 않습니다.

_기울임체_

~~취소선~~

```

## 헤드라인 만들기

# 제목 1단계
## 제목 2단계
### 제목 3단계
#### 제목 4단계
##### 제목 5단계
###### 제목 6단계
####### 헤드라인 효과 없음

```
# 제목 1단계
## 제목 2단계
### 제목 3단계
#### 제목 4단계
##### 제목 5단계
###### 제목 6단계
####### 헤드라인 효과 없음
```

제목에 크기 차이 및 위계를 주어 글을 문단별로 읽기 좋게 합니다.  
해시 또는 샤프(#)글쇠를 1개부터 6개까지 잇달아 쓴 뒤 띄어쓰기를 하고 넣으려는 제목을 씁니다.  

## 제목 표시하기

큰 제목
======

작은 제목
--------

```
큰 제목
======

작은 제목
--------
```

제목으로 표시하려는 문장 바로 아래에 = 글쇠 또는 - 글쇠를 연타해 넣습니다.

## 하이퍼링크 앵커

https://www.naver.com  
[네이버 메인 페이지](https://www.naver.com)를 다음처럼 다른 단어 속 하이퍼링크로 연결시킬 수 있습니다.

```
https://www.naver.com  
[네이버 메인 페이지](https://www.naver.com)를 다음처럼 다른 단어 속 하이퍼링크로 연결시킬 수 있습니다.
```

기본적으로 양식을 갖춘 도메인 네임은 별도의 처리 없이 자동으로 해당 페이지로 연결됩니다.  
도메인 네임 바로 다음 다른 문자열이 끼면 띄어쓰기가 나올 때까지 도메인 네임의 연장으로 인식합니다.  
그러므로 도메인 네임을 인용한 후에 반드시 띄어쓰기 한 번으로 도메인 네임을 구분시킵시다.  

하이퍼링크 앵커를 설정하려는 문자열을 대괄호로 감싸고,  
그 옆에 소괄호로 감싼 도메인 네임을 넣으면 해당 문자열을 눌렀을 때 의도한 페이지로 연결시킬 수 있습니다.
